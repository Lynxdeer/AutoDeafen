# AutoDeafen

Automatically deafens when you reach a certain percentage in a level, then undeafens when you win/die.

Configurable per-level in the pause menu (disabled by default).
### Includes:
* A neat config menu in the pause menu
* Configurable Percentage per-level
* Configurable Global Enabling and Percentage (In Mod Config)
